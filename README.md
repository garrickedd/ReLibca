# ReLibca

## Used Tools

1. [Gin as framework](https://github.com/gin-gonic/gin)
2. [Postgresql as main database engine](https://github.com/postgres/postgres)
3. [SQLX for database interactive](https://github.com/go-gorm/gorm)
4. [Viper for configurations](https://github.com/spf13/viper)
5. [JWT for authentication and authorization](https://github.com/golang-jwt/jwt)
6. [Cloudinary for images saving](https://cloudinary.com/)
7. Docker compose for run project with all dependencies in docker
